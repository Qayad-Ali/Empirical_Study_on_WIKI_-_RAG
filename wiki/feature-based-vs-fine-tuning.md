## Summary
BERT can be used with either a fine-tuning approach (adding task-specific layers) or a feature-based approach (extracting fixed representations).

## Explanation
The fine-tuning approach jointly trains all parameters for a task, while the feature-based approach uses pre-trained representations as fixed features. Both methods achieve strong results, with the feature-based approach showing competitive performance on tasks like CoNLL-2003 NER. [[squad-results]]
